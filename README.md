# Calculadora de Gasto Energético Diário

Aplicação web desenvolvida para calcular o gasto energético diário com base na fórmula de Harris-Benedict, considerando dados pessoais e nível de atividade física.

## Demonstração
Acesse a aplicação em funcionamento:  
https://leocampelo-dev.github.io/Projeto_Calculadora_gasto_energ-tico/

## Funcionalidades
- Cálculo do metabolismo basal (Harris-Benedict)
- Seleção de sexo
- Entrada de peso, altura e idade
- Seleção do nível de atividade física
- Exibição do gasto energético diário estimado

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## Estrutura do Projeto
- `index.html` – Estrutura da aplicação
- `styles/` – Estilização e layout
- `script.js` – Lógica de cálculo e interações

## Objetivo do Projeto
Projeto desenvolvido com foco em prática de JavaScript no navegador, manipulação do DOM, organização de código e construção de interfaces funcionais sem frameworks.
